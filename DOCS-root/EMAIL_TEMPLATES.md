## How to Use These Templates

1. **Personalize Everything**
   - Replace [Name], [Company], [specific problem]
   - Reference their LinkedIn post, GitHub repo, tweet
   - Make it feel 1-on-1

2. **Track Responses**
   - Use a spreadsheet: Name, Email, Date Sent, Response
   - Follow up 2-3 days later if no response
   - Don't spam beyond 3 emails

3. **A/B Test**
   - Try different subject lines
   - Test different CTAs
   - Measure open rates + click rates

4. **Automate Later**
   - Start manual (personalized = higher conversion)
   - Once you have 50+ customers, automate sequences
   - Tools: Mailchimp, ConvertKit, or just Gmail

- **Open Rate:** 40%+ (good subject lines)
